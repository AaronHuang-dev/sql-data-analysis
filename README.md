# SQL Data Analysis – Intel Sustainability Impact Analysis

## Overview

This project analyzes Intel's 2024 device repurposing program using SQL to evaluate sustainability performance and identify opportunities to maximize environmental impact.

## Skills Demonstrated

- SQL
- INNER JOIN
- Common Table Expressions (CTEs)
- CASE Statements
- Aggregate Functions
- GROUP BY
- Business Intelligence
- Data Analysis

## Project Highlights

- Analyzed over 600,000 repurposed devices
- Measured energy savings and CO₂ reductions
- Identified sustainability trends by device type, age, and region
- Developed data-driven business recommendations
